How to join the Java server

Steps

Follow these steps to join the Minecraft Java edition server.

1

Check your game version

You can only join the server if you are on the Java edition of Minecraft. If you are on Java, make sure you load the game on the 1.16.4 or 1.16.5 update.
2

Adding the Server

Once the game is loaded, click on the multiplayer button. Then click on "add server". Enter jonahwill.org as the IP and Jonahwill as the name (make sure there are no spaces before the IP). Once you have done this, click on "Done".
3

Joining the Server

Once the server is added you should see it in your list! Click on the server and then on "Join Server", at the bottom, or, double-click on the correct server.
